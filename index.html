What it does

A real ingest pipeline, in a panel.

Imports on its own

Drop in footage and it appears in your project — in bins that mirror your folder tree.

Sequences as one clip

Numbered image frames (EXR, DPX, PNG…) come in as a single clip — not hundreds of stills.

Watch only what matters

Tick the subfolders you care about. The rest is ignored, and your folder names carry over as bins.

Offline, never gone

Delete a source file and its clip goes offline — your timeline cuts stay intact. Nothing is yanked.

No duplicates

Point it at an existing project and it adopts the clips already there instead of importing copies.

Resumes after a restart

Reopen Premiere and it picks up the same folder, the same bins, watching — automatically.

Questions

Good to know.

Will it duplicate my media?

No. It recognises clips already in your project and adopts them instead of importing a second copy.

Is it safe for a live project?

Yes — non-destructive. It only adds, refreshes, and takes missing media offline. It never deletes clips or bins.

Does it run in the background?

It syncs whenever Premiere is open and watching, and resumes the same folder automatically on relaunch.

Mac?

Windows for now (Premiere Pro 2020+). Mac support is in the works.

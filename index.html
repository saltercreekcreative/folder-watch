Questions

Good to know.

Will it duplicate my media?

No. It recognises clips already in your project and adopts them instead of importing a second copy.

Is it safe for a live project?

Yes — non-destructive. It only adds, refreshes, and takes missing media offline. It never deletes clips or bins.

Does it run in the background?

It syncs whenever Premiere is open and watching, and resumes the same folder automatically on relaunch.

Mac?

Windows for now (Premiere Pro 2020+). Mac support is in the works.
